https://www.npmjs.com/package/@sap/approuter



this is need to install.

npm i @sap/approuter


add this in your script section  -------->

"start": "node node_modules/@sap/approuter/approuter.js"

create a xs-app.json in the approuter folder.

create a webapp folder in that you can create any html or any other appliaction. to load first.

create a manifest.yaml to inform sap BTP to deploy all the services together.

then follow the APPROUTER_SETUP_GUIDE_AI_generated.md file    ----> we can do via manifest.yaml and mta.yaml deployment described both over there.
